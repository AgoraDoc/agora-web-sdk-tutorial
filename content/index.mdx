## 教程目标

本教程会结合音视频概念讲解、SDK 概念讲解及 SDK 集成实际操作，帮助你达到以下目标：
- 理解音视频通话、直播的原理以及声网的技术方案。
- 理解声网 RTC Web SDK 的 API 设计。
- 可以集成并调用声网 RTC Web SDK 实现一个简单的音视频通话应用。

## 使用前提

你需要一台连接了麦克风和摄像头的个人电脑，并且安装了最新版本的 Chrome、FireFox、Edge 或 Safari。关于 Agora 实时音视频 Web SDK 的浏览器支持，详见 [浏览器支持
](https://docs.agora.io/cn/Interactive%20Broadcast/web_sdk_compatibility)。
